# Student Grade Management System (SGMS)
A complete C++ application designed to manage student academic records using a self-balancing AVL Tree data structure. This project was developed following a structured process including Requirements, System Architecture, Program Design, Test Plan, and Final Deliverable documentation.

---

##  Features
The Student Grade Management System includes:

### Student Record Management
- Add new students  
- Search for students by ID  
- Update existing student information  
- Remove students by ID  
- Display all students sorted by Student ID

### File Persistence
- Save all student data to a CSV file  
- Load student data from a CSV file  
- Automatically rebuild AVL Tree on load

### Balanced AVL Tree
- Fast search, insert, update, and delete operations  
- Guaranteed O(log n) performance  
- Supports rotations (LL, RR, LR, RL)

### Clean Menu-Driven Interface
- Easy-to-use command-line interface  
- Input validation  
- Clear prompts and responses
